# Design — Procedure Feedback Review Dashboard

## Product Intent

Feature 039 captured local feedback around the `ProcedureWorkflow` outcome object. Feature 040 adds a review page so that feedback becomes inspectable product signal.

This is still a local/exportable MVP. It intentionally avoids backend persistence, identity, permissions, and remote telemetry.

## Page

```text
public/procedure-feedback-dashboard.html
```

## Data Source

```text
localStorage["la-muni-rag:procedure-feedback"]
```

Expected shape is the feedback payload generated by `public/procedure-feedback.js`:

```ts
type ProcedureWorkflowFeedback = {
  id: string;
  createdAt: string;
  workflowId: string;
  workflowTitle: string;
  procedureType: string;
  jurisdiction: string;
  confidence: string;
  query: string;
  stepNumber: string;
  stepTitle: string;
  feedbackType: string;
  comment: string;
};
```

## UI Sections

1. Hero/status panel.
2. Metrics row.
3. Filters:
   - feedback type;
   - free-text search.
4. Feedback card list.
5. Actions:
   - copy filtered JSON;
   - copy all JSON;
   - clear local feedback.
6. Governance note:
   - feedback is local;
   - feedback is not municipal evidence;
   - do not paste sensitive data.

## Safety

- Dynamic content is escaped before rendering.
- No network calls.
- Clear action requires explicit confirmation.
- The page must work with empty or malformed localStorage.

## Pages Build

`build-pages.mjs` patches root-relative links for the new dashboard route.

`verify-pages-artifact.mjs` requires:

```text
procedure-feedback-dashboard.html
```

and checks that static artifact links are project-page safe.

## Test Strategy

Static tests verify:

- dashboard page exists and reads the correct localStorage key;
- no network APIs are used;
- metrics, filters, feedback cards, export, and clear affordances exist;
- dynamic text is escaped;
- procedure page links to the dashboard;
- Pages scripts include and verify the dashboard;
- documentation explains local-only governance and future backend path.
